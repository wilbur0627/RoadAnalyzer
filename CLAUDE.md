## Custom Commands

- `snap`: 當我下達此指令，請依照以下規範更新 `ME.md`, 如果沒有該檔案請創建：
  1. **動態結構掃描**：執行 `tree -L 2`，必須自動過濾 `.claudeignore` 定義的所有路徑。
  2. **嚴禁主動讀取**：除非我明確要求，否則嚴禁 `cat` (讀取) 檔案內容。請優先利用當前對話記憶來提取功能變更點。
  3. **邏輯過濾 (非 ignore 檔案)**：
  4. **更新內容**：
     - 更新 `ME.md` 中的「目前架構總覽」。
     - 準確記錄新增的 API 端點或檔案內容，描述大概的功能。
  5. **極致省錢**：完成更新後，僅回覆「已存檔」，不要重複列出已寫入的細節。
